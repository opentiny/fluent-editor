# 表格

TinyEditor 的表格功能依赖 `quill-table-up`，使用前请先安装 `quill-table-up`。

```bash
npm install quill-table-up
```

具体相关参数请查看[文档](https://github.com/zzxming/quill-table-up?tab=readme-ov-file#usage)
